# PWA 설치 QA

이 문서는 파일럿 운영자가 Android와 iOS에서 MOVE RE PWA 설치를 직접 확인할 때 사용합니다. 설치 테스트는 앱 진입 가능성을 확인하기 위한 것이며, 설치 후에는 매번 설치 테스트를 반복하지 않고 홈 화면 앱으로 진입해야 합니다.

## 공통 주의사항

- [ ] 카카오, 인스타그램, 네이버 등 인앱 브라우저에서는 PWA 설치가 제한될 수 있습니다.
- [ ] 인앱 브라우저에서 문제가 보이면 외부 브라우저로 열도록 안내합니다.
- [ ] Android는 Chrome 또는 Samsung Internet을 우선 사용합니다.
- [ ] iOS는 Safari를 사용합니다.
- [ ] 설치 후 진입 경로는 `/app/home`입니다.
- [ ] PWA standalone 상태에서는 브라우저 주소창이 보이지 않는 것이 정상입니다.
- [ ] 설치가 완료된 사용자는 다시 설치 CTA를 찾는 것이 아니라 홈 화면 앱 아이콘으로 진입해야 합니다.

## Android 설치 QA

### 준비

- [ ] Android 실기기 준비
- [ ] Chrome 최신 버전 확인
- [ ] 기존 MOVE RE PWA가 있으면 필요한 경우 제거 후 재테스트
- [ ] Production URL을 Chrome 일반 탭에서 열기

### 확인 절차

- [ ] Chrome에서 MOVE RE 접속
- [ ] 로그인 또는 파일럿 접근 흐름을 거쳐 `/app/home` 진입
- [ ] PWA 설치 카드가 보이는지 확인
- [ ] 설치 방법 보기 또는 브라우저 설치 안내가 정상적으로 보이는지 확인
- [ ] 설치가 직접 되지 않는 환경에서 링크 복사 fallback 안내가 가능한지 확인
- [ ] Chrome 메뉴 또는 설치 프롬프트로 홈 화면에 앱 추가
- [ ] 홈 화면의 MOVE RE 앱 아이콘으로 실행
- [ ] standalone 상태로 실행되는지 확인
- [ ] standalone 상태에서 설치 카드가 숨겨지는지 확인
- [ ] `/app/home`에서 리셋맵 또는 세션 상태가 정상 표시되는지 확인

### 실패 시 빠른 판단

- Chrome이 아닌 인앱 브라우저라면 외부 Chrome으로 열도록 안내합니다.
- 이미 설치되어 있는데 설치 카드가 보이지 않는 것은 정상일 수 있습니다.
- 사용자가 설치 후에도 Chrome 탭을 보고 있다면 홈 화면 앱 아이콘으로 다시 열도록 안내합니다.

## iOS 설치 QA

### 준비

- [ ] iPhone 실기기 준비
- [ ] Safari 사용 확인
- [ ] 기존 MOVE RE 홈 화면 앱이 있으면 필요한 경우 제거 후 재테스트
- [ ] Production URL을 Safari 일반 탭에서 열기

### 확인 절차

- [ ] Safari에서 MOVE RE 접속
- [ ] 로그인 또는 파일럿 접근 흐름을 거쳐 `/app/home` 진입
- [ ] iOS에서는 Android 같은 native install prompt가 뜨지 않는다는 점 확인
- [ ] Safari 공유 버튼을 탭
- [ ] "홈 화면에 추가" 선택
- [ ] 홈 화면에 MOVE RE 앱 아이콘이 생성되는지 확인
- [ ] 홈 화면 앱 아이콘으로 실행
- [ ] Safari 일반 탭이 아니라 PWA standalone으로 열리는지 확인
- [ ] standalone 상태에서 `/app/home` 진입 확인
- [ ] Safari 일반 탭과 PWA standalone의 차이를 운영자가 설명할 수 있는지 확인

### Safari 일반 탭과 PWA standalone 차이

- Safari 일반 탭은 주소창이 보이고, 설치 전 탐색 환경입니다.
- PWA standalone은 홈 화면 앱 아이콘으로 실행되며 주소창이 보이지 않습니다.
- push permission card와 테스트 알림 QA는 PWA standalone 기준으로 확인합니다.
- Safari 일반 탭에서 알림 카드가 보이지 않는 것은 정상입니다.

## 설치 후 확인

- [ ] `/app/home`으로 진입되는지 확인
- [ ] 리셋맵이 보이는지 확인
- [ ] 세션 패널을 열 수 있는지 확인
- [ ] 첫 세션을 시작할 수 있는지 확인
- [ ] 설치 안내가 반복적으로 노출되지 않는지 확인
- [ ] 사용자가 다음 방문부터 홈 화면 앱으로 진입할 수 있는지 안내

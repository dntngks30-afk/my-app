# Acceptance Tests - Movement Test

## A. /movement-test
- [ ] "테스트 시작" 버튼이 보인다.
- [ ] 버튼 클릭 시 /movement-test/survey 로 이동한다.
- [ ] (선택) 기존 세션이 있으면 "이어하기"로 표시되거나 안내가 나온다.
- [ ] (선택) "처음부터 다시" 실행 시 설문 데이터가 초기화된다.

## B. /movement-test/survey 진행/저장
- [ ] 첫 진입 시 1페이지(5문항)가 보인다.
- [ ] 답변 선택 즉시 새로고침해도 선택이 유지된다.
- [ ] 다음 버튼은 현재 페이지 질문을 모두 답해야 활성화된다.
- [ ] 이전 버튼으로 돌아가도 답변이 유지된다.
- [ ] 중간에 앱 종료 후 재진입해도 마지막 진행 페이지로 복구된다.

## C. 제출/결과
- [ ] 마지막 페이지에서 "제출"이 노출된다.
- [ ] 제출 클릭 시 /movement-test/result로 이동한다.
- [ ] 결과가 항상 1개로 확정되어 표시된다(동점이어도).
- [ ] 결과 페이지 새로고침해도 동일 결과가 유지된다.

## D. 결과 페이지 진입 예외
- [ ] 설문 없이 /movement-test/result 직접 진입 시 "데이터 없음" 안내가 나온다.
- [ ] 안내 버튼을 누르면 /movement-test로 이동한다.
